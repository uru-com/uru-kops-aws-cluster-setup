# Documentation Overview

This folder contains the main documentation for the Kops AWS cluster setup project.

## Included Docs

- `cluster-creation-guide.md` — step-by-step instructions for creating a Kubernetes cluster with Kops on AWS.
- `diagram-descriptions.md` — explanations for the architecture and workflow diagrams in the repo.

## How to use

1. Start with `README.md` at the repository root.
2. Open `docs/cluster-creation-guide.md` for the full provisioning workflow.
3. Review `docs/diagram-descriptions.md` for visual context on each architecture image.
