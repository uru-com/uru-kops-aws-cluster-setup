# Contributing

Thanks for your interest in improving this repository.

## How to contribute

- Add new diagrams to `images/` and reference them from `README.md` or docs files.
- Update `docs/cluster-creation-guide.md` with new commands, best practices, or AWS configuration notes.
- Keep image paths relative so GitHub renders them correctly.
- Use clear headings, code blocks, and tables for readability.

## Docs workflow

1. Edit `docs/cluster-creation-guide.md` for procedural updates.
2. Edit `docs/diagram-descriptions.md` when diagrams change.
3. Verify the root `README.md` still renders correctly on GitHub.
4. Open a pull request with a short summary of changes.
